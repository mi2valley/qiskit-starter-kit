# Qiskit Serverless Starter Kit

## Requirements

- python 3.12
- poetry
- docker

## Quick Start

```bash
% git clone git@github.com:mi2valley/qiskit-starter-kit.git
% cd qiskit-starter-kit
% git fetch origin qiskit-serverless
% git checkout -b qiskit-serverless origin/qiskit-serverless
% poetry install
% docker compose up
```
